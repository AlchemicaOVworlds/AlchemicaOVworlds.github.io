How we made new world

How the Game works

New World is an MMORPG developed by Amazon Games that utilizes both client-side and server-side technology to deliver a seamless gaming experience. On the server side, the game runs on Amazon Web Services (AWS), a cutting-edge cloud network infrastructure that allows the studio to host up to 10,000 players per server. The game’s developers used a unique architecture built around different instances of Amazon Elastic Compute Cloud (Amazon EC2), which provides secure and resizable compute capacity for virtually any workload.

On the client side, there is a lot going on graphically to deliver an immersive gaming experience 3. The game’s client technology is responsible for rendering the game’s graphics, processing user input, and communicating with the game’s servers.

Together, these technologies allow New World to deliver a seamless and immersive gaming experience to players. The game’s servers handle the complex computations required to simulate the game’s virtual world, while the client technology delivers high-quality graphics and responsive gameplay.

How our Virtual World Is being used

New World is a massively multiplayer online role-playing game (MMORPG) developed by Amazon Games and released in September 2021. In the game, players create custom characters and enter an expansive, supernatural realm called Aeternum, where they can interact with thousands of other virtual adventurers.

The game’s developers used Amazon Web Services (AWS) solutions to create a seamless user experience while maintaining and operating a backend structure that keeps the game available virtually 24/7 globally. The game’s architecture is built around different instances of Amazon Elastic Compute Cloud (Amazon EC2), which provides secure and resizable computing capacity for virtually any workload.

As players move through the game, state information is handed from hub instance to hub instance, spreading the compute load over virtual servers hosted globally in multiple AWS Regions. This allows for a seamless and immersive experience for players as they explore the virtual world of Aeternum.

In addition to exploring the virtual world, players can also participate in session-based modes, where they can take part in temporary adventures. These sessions are powered by a separate pool of hubs shared based on availability.

Overall, the use of virtual worlds in New World allows for an immersive and engaging gaming experience, where players can explore a vast and detailed world, interact with other players, and engage in a variety of activities and adventures.

Social and ethical Issues

Copyright is a legal right that grants the creator of an original work exclusive rights to its use and distribution. When creating an MMORPG, it is vital to ensure that all content used in the game, including graphics, music, and code, is either created by the game developers or properly licensed by the original creators. Failure to do so can result in legal action and financial penalties.

Privacy is the ability of an individual or group to seclude themselves or information about themselves from others. When creating an MMORPG, it is important to ensure that players’ personal information is protected and not shared without their consent. This includes information such as their real name, address, and payment information. It is also important to have clear privacy policies in place and to follow all relevant privacy laws.

Accuracy of data refers to the correctness and reliability of the data being used. When creating an MMORPG, it is important to ensure that any data used in the game, such as player statistics or game mechanics, is accurate and reliable. This can help to ensure a fair and enjoyable gaming experience for all players.

In addition to the above, it is also important to consider the following when creating an MMORPG:

  • User experience: The user experience (UX) is the overall experience that a user has when interacting with a product or service. When creating an MMORPG, it is important to focus on creating a positive UX for players. This includes things like making the game easy to learn and navigate, providing clear and concise instructions, and offering a variety of features and activities that players can enjoy.
  • Game balance: Game balance is the state of a game in which all of the elements are working together in a fair and competitive way. When creating an MMORPG, it is important to carefully balance the game's mechanics, items, and characters. This will help to ensure that all players have a chance to succeed and that the game is enjoyable for everyone.
  • Content updates: MMORPGs are constantly evolving games, and it is important to keep the content fresh and exciting for players. This can be done by adding new features, areas, and quests to the game on a regular basis.
  • Community: A strong community is essential for any MMORPG. When creating an MMORPG, it is important to foster a positive and welcoming community for players. This can be done by providing forums, chat rooms, and other social features where players can interact with each other.
